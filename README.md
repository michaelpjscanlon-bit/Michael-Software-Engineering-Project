# Michael Software Engineering Project

This is a Flask-based NBA statistics website created for my Software Engineering project.

The system uses Python, Flask, HTML, CSS, SQLite and graph visualisation to display and compare NBA statistics.

## Main features

- NBA statistics website
- Award pages such as MVP, DPOY, FMVP and 6MOT
- Team and player data pages
- Graph-based visualisation
- Flask backend connected to a local SQLite database

## Database note

The full NBA SQLite database was not uploaded to GitHub because it exceeded GitHub's file size limit. The database is stored locally and was used during testing and demonstration.
